Module Hierarchy

  • pywm: Main module for the PYWM window manager.

Class Hierarchy

  • pywm.Fl_Button: Creates and manages button widgets for use in Fl_Window popup windows.
  • pywm.Fl_Hold_Browser: Creates and manages listbox widgets for use in Fl_Window popup windows.
  • pywm.Fl_Select_Browser: Creates and manages listbox widgets for use in Fl_Window popup windows.
  • pywm.Fl_Window: This class allows you to create popup windows for user interaction.
  • pywm.WM: pywm.WM is the main PYWM class that you should subclass when building your own custom window manager.
  • pywm.window: This class wraps an X window in a form where your python window manager code can manipulate it.
